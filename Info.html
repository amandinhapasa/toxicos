Sobre o Site

Neste site, você aprenderá sobre o uso de agrotóxicos e encontrará um código do Arduino IDE (uma plataforma de software utilizada para programar placas de microcontroladores Arduino) para utilizar no seu equipamento agrícola, permitindo controlar a liberação dos produtos.
